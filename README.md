How to work with:

1 - Create a new virtual environment:
```bash
virtualenv env
```

2 - Start virtualenv:
```bash
source env/bin/activate
```

In repository root file:
3 - 
```bash
pip install -r requirements.txt
```

4 - Open jupyter notebook:
```bash
jupyter notebook
```

5 - Start working :)
